# project

## Description

Start this test repo with npm start and then hit the URL http://localhost:3000. The logs then shows entering and exiting at the top level only and not for the service method.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
